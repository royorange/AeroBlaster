# 美术生成 Prompt 体系

> **目的**：本文档是 AeroBlaster 全套美术资产的生成手册。所有 AI 生成的图必须遵守同一套"美术圣经"，以保证风格一致。
>
> **使用方法**：每次生成新素材时，**永远先上传"美术圣经参考图"作为 reference image**，再从下面对应章节复制 prompt 主体。

---

## 0. 美术圣经（永远附带这段）

### 0.1 风格定调

**Neo-Retro Mecha** — 高分辨率"伪像素"质感 + 机甲重装感 + 霓虹冷色

### 0.2 风格基准段（所有 prompt 开头都要写）

```
=== 视觉风格基准 ===
严格参照所附 TEMPEST 战机参考图（玩家阵营基础态）的美术风格、细节密度、
配色饱和度、光照处理、CRT 纹理强度，保持完全一致的视觉语言。

风格关键词：Neo-Retro Mecha 顶视科幻机甲、高分辨率"伪像素"质感、
硬边机械细节、自发光 + Bloom、纯黑背景、强对比配色。

技术规范：
- 顶视图 90 度角（俯视），主体方向朝上
- 自发光元素配 Bloom 光晕和 rim light
- 微弱的 CRT 扫描线纹理叠加
- 纯黑背景 (#000000)，无场景无环境
```

### 0.3 配色规范（强制遵守）

**玩家阵营（冷色 + 多彩）**：
- INFERNO 烈焰: 纯黑 (#1a1a1a) + 暗橙红 (#a83a1a) + 炽橙发光 (#ff6a1a)
- TEMPEST 风暴: 深紫 (#3a0a5a) + 银色边缘 + 亮紫电弧 (#e040ff)
- BASTION 堡垒: 枪灰 (#3a3a4a) + 金色装甲条 (#d4a520) + 白色能量核心

**敌方阵营（暖色，统一）**：
- 主装甲: 深暗红 (#5a0a14) + 黑色金属 (#0a0a0a)
- 弱点核心: 炽烈橙红 (#ff3a1a)
- 装甲条: 暗橙 (#8a3a0a)

**通用元素**：
- 命中/危险: 橙红 (#ff7a1a)
- 暴击/特殊: 洋红 (#ff2dcd)

### 0.4 标签样式（所有有标签的图统一）

```
- 大字（青色发光英文，粗体科幻字体）：JOB_NAME
- 中字（白色中文）：中文名
- 小字（灰色中文）：机制说明 · 英文小字
```

### 0.5 严格禁止（所有 prompt 都加）

```
=== 严格禁止 ===
- 不要触手 / 有机曲线 / 生物元素（保持机械机甲）
- 不要彩虹色 / 粉彩色 / 大量饱和度高的杂色
- 不要纯像素（8-bit）风
- 不要 20 年前街机风
- 不要写实烟雾粒子（要几何感）
- 玩家阵营和敌方阵营色绝不能撞
```

## 1. 三职业飞机本体（基础态）

> **现状**：已完成，三职业基础态对比图已生成。后续若单独生成请用下面模板。

### 1.1 单独生成某一职业（透明背景，方便切图入引擎）

适用：把对比图里的某一架飞机重新生成为独立 sprite，便于直接放入游戏。

```
=== [复制 0.2 风格基准段] ===
=== [复制 0.5 严格禁止] ===

=== 任务 ===
为顶视垂直射击 roguelike 游戏设计单架战机的基础态独立 sprite。
**透明背景 PNG**，居中构图，方便后续切入游戏引擎。

=== 设计要求（按职业选一段使用）===

【INFERNO 烈焰】
- 紧凑子弹形机身，强烈纵向流线感
- 机首前方平台只装基础的 2 根重炮管（升级后会动态加炮管到 8 根）
- 机身两侧"龙鼻孔"散热口（橙红色发光，过载机制可视化）
- 顶部圆形过热核心（暗红脉动）
- 配色：纯黑装甲 (#1a1a1a) + 暗橙红装甲条 (#a83a1a) + 炽橙发光 (#ff6a1a)
- 引擎：橙黄双引擎尾焰
- 视觉重量：90%
- 比喻：紧凑的火焰炮艇

【TEMPEST 风暴】
- 紫色晶体战机美学（直接参照附图）
- 不对称几何机身
- 中央菱形晶体核心暗淡（待激活，电荷资源可视化）
- **基础态没有浮游电球**（升级闪电球词条后会动态出现 1-5 个）
- 机身两侧留出挂载浮游物的空间
- 配色：深紫 (#3a0a5a) + 银色边缘 + 暗紫核心
- 引擎：紫色等离子双引擎
- 视觉重量：80%

【BASTION 堡垒】
- 厚重战舰形机身，宽大装甲板
- 中央一根大型主炮（轨道炮），但 Lv.1 时是暗淡未充能状态
- 双副炮简洁，无装饰花纹
- 机身两侧留出挂载副炮塔/导弹舱的空间
- 配色：枪灰 (#3a3a4a) + 金色装甲条 (#d4a520) + 中央能量核心暗淡白色
- 引擎：深红色厚重双引擎
- 视觉重量：130%
- 比喻：缓慢的攻城炮舰

=== 关键约束 ===
- 飞机本体在游戏中不会变形态，要"留出挂载空间"
- 机首前方留位置加炮管 / 机身两侧留位置加副件 / 机身周围留位置加浮游物
- 缩到 64x64 缩略图剪影仍可分辨

=== 排版 ===
- 单张 1024x1024 透明背景 PNG (alpha channel)
- 飞机居中，占画布 60-70%
- 无标签、无边框、无背景元素
- 引擎尾焰可以延伸到画布底部
```

## 2. 武器挂件 Sprite 库

> **目的**：词条升级武器时，动态挂载到飞机上的小 sprite。每个武器在不同等级有不同视觉。

```
=== [复制 0.2 风格基准段] ===
=== [复制 0.5 严格禁止] ===

=== 任务 ===
为顶视垂直射击 roguelike 设计武器挂件 sprite sheet。这些武器会在游戏内
动态挂载到玩家飞机上（飞机本体不变，玩家通过收集词条升级武器）。

每个武器有 5 个升级阶段（Lv.1 → Lv.5），从基础到满配。
所有武器为顶视图，方向朝上（炮口朝上）。

=== 武器列表（横向 5 列对应 Lv.1-5）===

【INFERNO 主武器：散射炮】
- Lv.1: 1 根短炮管，橙色炮口
- Lv.2: 2 根并排短炮管
- Lv.3: 3 根并排短炮管，中间稍长
- Lv.4: 5 根并排短炮管，开始扇形排列（小角度）
- Lv.5: 8 根扇形排列重炮管，全部炽烈发光，散热鳍片可见

【TEMPEST 主武器：浮游闪电球】
- Lv.1: 1 个紫色小球，发出微弱电弧
- Lv.2: 2 个紫色球，电弧加粗
- Lv.3: 3 个紫色球，连接电弧形成三角网
- Lv.4: 4 个球围绕中心，电弧密集
- Lv.5: 5 个球 + 中心核 + 全身电弧网，紫色雷霆光晕

【BASTION 主武器：充能轨道炮】
- Lv.1: 单管轨道炮，暗淡白色能量核
- Lv.2: 双管轨道炮，半亮能量核
- Lv.3: 双管 + 副炮塔×2
- Lv.4: 三管轨道炮，金色装饰
- Lv.5: 三管 + 4 副炮塔，主炮拉出强烈白色光柱

【副武器：追踪导弹（通用解锁）】
- Lv.1: 单导弹舱（侧挂式），1 枚导弹
- Lv.2: 双导弹舱
- Lv.3-5: 多导弹舱 + 装填指示灯

【副武器：穿透激光（通用解锁）】
- Lv.1-5: 激光发射器越来越大，前端能量聚焦点越来越亮

=== 排版 ===
- 单张 2400x1600 横版画布，纯黑背景 (#000000)
- 5 行（5 种武器） × 5 列（Lv.1-5）
- 每个 sprite 在自己的网格内居中
- 行左侧标签：散射炮 / 闪电球 / 轨道炮 / 追踪导弹 / 穿透激光
- 列顶部标签：Lv.1 / Lv.2 / Lv.3 / Lv.4 / Lv.5

=== 关键约束 ===
- 每个武器 sprite 独立，便于程序切割
- 同一武器的 5 个等级保持中心对齐和同样的"挂载点"位置
- 武器配色按职业（散射炮橙、闪电球紫、轨道炮金白）
```

## 3. 子弹 / 命中特效 Sprite Sheet

```
=== [复制 0.2 风格基准段] ===
=== [复制 0.5 严格禁止] ===

=== 任务 ===
为顶视垂直射击游戏生成子弹与命中特效的 sprite sheet。

=== 内容（共 8 个面板，2 行 4 列）===

第一行 — 子弹/弹道（玩家阵营，冷色）：
1. 基础子弹：小型青色等离子弹，带短尾迹
2. 散射子弹（INFERNO）：橙色火焰弹，带燃烧粒子尾迹
3. 链式电流（TEMPEST）：紫色锯齿状电弧，连接两点
4. 充能轨道炮（BASTION）：白色穿透光束，金色边缘 + 蓝白核心

第二行 — 命中/状态特效：
5. 小型命中：青色环形冲击波 + 火花迸发（4 帧序列横排）
6. 燃烧持续：橙色火焰图标 + 烟雾上升（4 帧循环动画）
7. 中型爆炸：橙红环形扩散 + 六边形碎片（4 帧序列）
8. 暴击爆炸：洋红色星爆 + 同心圆光环 + "CRIT!"字样（4 帧序列）

=== 设计原则 ===
- 几何感而非真实烟雾感（参考 Geometry Wars / Vampire Survivors）
- 硬边 + Bloom 光晕
- 矢量般干净的形状（环、六边形、星爆）
- 配色严格分阵营：玩家青/紫/白/金，敌方橙/红
- 每帧之间动作清晰

=== 排版 ===
- 单张 2400x1280 横版画布，纯黑背景 (#000000)
- 8 个面板，每个有清晰边框分隔
- 每个面板下方标签：
  - 中文名（白色）：基础子弹 / 散射弹 / 链电 / 轨道炮 / 命中 / 燃烧 / 爆炸 / 暴击
  - 英文小字（青色）：BASIC / SPREAD / CHAIN / RAILGUN / HIT / BURN / EXPLODE / CRIT
- 多帧动画的面板内部，帧从左到右排列
```

## 6. 敌人三档分级

```
=== [复制 0.2 风格基准段] ===
=== [复制 0.5 严格禁止] ===

=== 任务 ===
为顶视垂直射击 roguelike 游戏设计敌方阵营的三档敌人对比图。
玩家阵营是冷色调（紫/蓝/银），敌方阵营必须用暖色调（红/橙）建立"敌我清晰"的视觉对立。

  - Grunt-A 直线型：标准对称三角形，单炮
  - Grunt-B 蛇形：有侧翼，机身略宽
  - Grunt-C 追踪型：有锁定瞄准器细节
  - Grunt-D 射手：双炮管，机身略大

【档 2】精英敌机 ELITE（3 种变体，中间一行）

- 不对称武器挂载，更多装甲板

- 缩到 96x96 仍可分辨

  - Elite-A 弹幕型：多炮塔环绕
  - Elite-B 突进型：尖锐前装甲，撞击意图明显
  - Elite-C 召唤型：背载小型敌机舱

【档 3】Boss 敌机（1 个，最下方居中放大）
- 巨型战列舰级（约 grunt 的 4 倍尺寸）

=== 配色（敌方阵营，强制）===
```

## 7. UI 元素

```
=== [复制 0.2 风格基准段] ===
=== [复制 0.5 严格禁止] ===

=== 任务 ===
为手游 roguelike 射击游戏设计 UI 框架元素。

=== 内容 ===

游戏内 HUD（720x1280 竖版）：
- 顶部：HP 条（六边形分段，红色）+ 资源条（按职业变色）
- 顶部右：分数 + 金币
- 底部：大招按钮（六边形冷却环）+ 小地图（如有）

主菜单 UI 元素：

- 全部使用六边形 / 切角矩形（科幻军事感）

=== 排版 ===
- 单张 2400x2400，分上下两区
- 上区：HUD mockup（720x1280 竖版预览）
- 下区：UI 元素库（按钮、卡片等独立 sprite）
```
