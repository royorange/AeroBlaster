## 快速开始

### 你要做什么？

| 你想生成 | 跳到章节 |

| 把 3 职业飞机切成单独透明 PNG（最优先） | [§1 飞机本体](#1-飞机本体单独透明背景) |
| 武器升级各阶段的 sprite（散射炮 / 闪电球 / 轨道炮） | [§2 武器挂件](#2-武器挂件-sprite-库) |
| 子弹、爆炸、命中特效 | [§3 子弹与命中特效](#3-子弹与命中特效) |

| 大招全屏特效 | [§5 大招全屏特效](#5-大招全屏特效) |
| 敌人小怪 / 精英 / Boss | [§6 敌人三档](#6-敌人三档) |
| UI 按钮、HUD、卡片框 | [§7 UI 元素](#7-ui-元素) |

### 推荐生成顺序

按这个顺序做，前面的产出会指导后面：

1. **§1 飞机本体** — 切出 3 张可入引擎的 PNG（最先做，因为后续武器要挂在飞机上）
2. **§2 武器挂件** — 词条升级的核心视觉（中等优先级）
3. **§6 敌人三档** — 战斗的另一半内容
4. **§3 子弹特效** — 战斗动效
5. **§4 被动光环** — 词条 UI
6. **§5 大招特效** — 高潮时刻
7. **§7 UI 元素** — 最后做，因为 UI 设计依赖前面的视觉风格

### 通用工作流

每次生成新素材时：

```
1. 打开 Image 2 / Nano Banana 2 / GPT Image
2. 上传"美术规范参考图"作为 reference image（见下面）
3. 找到对应章节，复制完整 prompt 文字
4. 跑图，生成 4-6 张挑最满意的
5. 切图入引擎，放到 assets/arts/ 对应目录
```

### 美术规范参考图

**所有 prompt 跑图前都必须上传这张图作为 reference**：

## §1 飞机本体（单独透明背景）

**目标**：把对比图里的 3 架飞机分别生成为独立透明 PNG，可直接放入 Cocos 引擎。

**做几次**：3 次，每次只生成一架（INFERNO / TEMPEST / BASTION 各一次）

### §1.1 INFERNO 烈焰（独立 PNG）

**完整 prompt（复制这一整段）**：

```
=== 视觉风格 ===
严格参照所附参考图（三职业飞机基础态）的美术风格、细节密度、配色饱和度、
光照处理、CRT 纹理强度，保持完全一致的视觉语言。

风格关键词：Neo-Retro Mecha 顶视科幻机甲、高分辨率"伪像素"质感、
硬边机械细节、自发光 + Bloom、强对比配色。

技术规范：
- 顶视图 90 度角（俯视），机首方向朝上
- 自发光元素配 Bloom 光晕和 rim light
- 微弱的 CRT 扫描线纹理叠加

=== 任务 ===
为顶视垂直射击 roguelike 游戏设计 INFERNO 烈焰战机的基础态独立 sprite。
**透明背景 PNG**，居中构图，方便后续切入游戏引擎。

=== INFERNO 设计要求 ===
- 紧凑子弹形机身，强烈纵向流线感
- 机首前方平台只装基础的 2 根重炮管（升级后会动态加炮管到 8 根）
- 机身两侧"龙鼻孔"散热口（橙红色发光，过载机制可视化）
- 顶部圆形过热核心（暗红脉动）
- 配色：纯黑装甲 (#1a1a1a) + 暗橙红装甲条 (#a83a1a) + 炽橙发光 (#ff6a1a)
- 引擎：橙黄双引擎尾焰
- 视觉重量：90%（中等）
- 比喻：紧凑的火焰炮艇

=== 关键约束 ===
- 飞机本体在游戏中不会变形态，要"留出挂载空间"
- 机首前方留位置加炮管 / 机身两侧留位置加副件
- 缩到 64x64 缩略图剪影仍可分辨

=== 排版 ===
- 单张 1024x1024 透明背景 PNG (alpha channel)
- 飞机居中，占画布 60-70%
- 无标签、无边框、无背景元素
- 引擎尾焰可以延伸到画布底部

=== 严格禁止 ===
- 不要触手 / 有机曲线 / 生物元素
- 不要彩虹色 / 粉彩色
- 不要纯像素（8-bit）风
- 不要场景或环境（必须透明背景）
- 不要任何文字标签
```

## §2 武器挂件 sprite 库

**目标**：词条升级武器时，动态挂载到飞机上的小 sprite。每个武器在不同等级有不同视觉。

**做几次**：1 次（一次性出 5 武器 × 5 等级 = 25 个 sprite 的网格大图，后续切图）

### §2.1 武器升级 sprite sheet

**完整 prompt（复制这一整段）**：

```
=== 视觉风格 ===
严格参照所附参考图（三职业飞机基础态）的美术风格、细节密度、配色饱和度、
光照处理、CRT 纹理强度，保持完全一致的视觉语言。

风格关键词：Neo-Retro Mecha 顶视科幻机甲、高分辨率"伪像素"质感、
硬边机械细节、自发光 + Bloom、强对比配色。

技术规范：
- 顶视图 90 度角，所有武器方向朝上（炮口朝上）
- 自发光元素配 Bloom 光晕和 rim light

=== 任务 ===
为顶视垂直射击 roguelike 设计武器挂件 sprite sheet。这些武器会在游戏内
动态挂载到玩家飞机上（飞机本体不变，玩家通过收集词条升级武器）。

每个武器有 5 个升级阶段（Lv.1 → Lv.5），从基础到满配。
所有武器为顶视图，方向朝上（炮口朝上）。

=== 武器列表（5 行 × 5 列）===

【第 1 行】INFERNO 主武器：散射炮
- Lv.1: 1 根短炮管，橙色炮口
- Lv.2: 2 根并排短炮管
- Lv.3: 3 根并排短炮管，中间稍长
- Lv.4: 5 根并排短炮管，开始扇形排列（小角度）
- Lv.5: 8 根扇形排列重炮管，全部炽烈发光，散热鳍片可见

【第 2 行】TEMPEST 主武器：浮游闪电球
- Lv.1: 1 个紫色小球，发出微弱电弧
- Lv.2: 2 个紫色球，电弧加粗
- Lv.3: 3 个紫色球，连接电弧形成三角网
- Lv.4: 4 个球围绕中心，电弧密集
- Lv.5: 5 个球 + 中心核 + 全身电弧网，紫色雷霆光晕

【第 3 行】BASTION 主武器：充能轨道炮
- Lv.1: 单管轨道炮，暗淡白色能量核
- Lv.2: 双管轨道炮，半亮能量核
- Lv.3: 双管 + 副炮塔×2
- Lv.4: 三管轨道炮，金色装饰
- Lv.5: 三管 + 4 副炮塔，主炮拉出强烈白色光柱

【第 4 行】副武器：追踪导弹（通用解锁）
- Lv.1: 单导弹舱（侧挂式），1 枚导弹
- Lv.2: 双导弹舱
- Lv.3: 双导弹舱 + 装填指示灯
- Lv.4: 三导弹舱
- Lv.5: 四导弹舱 + 中央指挥模块

【第 5 行】副武器：穿透激光（通用解锁）
- Lv.1: 小型激光发射器
- Lv.2: 中型激光发射器
- Lv.3: 大型激光发射器，能量聚焦点明显
- Lv.4: 双联装激光器
- Lv.5: 三联装激光器，前端能量聚焦炽烈

=== 排版 ===
- 单张 2400x2400 方形画布，纯黑背景 (#000000)
- 5 行 × 5 列网格（每个格子 480x480）
- 每个 sprite 在自己的网格内居中
- 行左侧标签（青色发光英文 + 白色中文）：
  Row 1: SPREAD CANNON / 散射炮
  Row 2: ORBIT ORBS / 闪电球
  Row 3: RAILGUN / 轨道炮

### §3.1 子弹与特效 sprite sheet

```
=== 视觉风格 ===
严格参照所附参考图（三职业飞机基础态）的美术风格、配色饱和度、Bloom 强度。

=== 任务 ===
为顶视垂直射击游戏生成子弹与命中特效的 sprite sheet。

=== 内容（共 8 个面板，2 行 4 列）===

第一行 — 子弹/弹道（玩家阵营，冷色）：
1. 基础子弹：小型青色等离子弹，带短尾迹
2. 散射子弹（INFERNO）：橙色火焰弹，带燃烧粒子尾迹
3. 链式电流（TEMPEST）：紫色锯齿状电弧，连接两点
4. 充能轨道炮（BASTION）：白色穿透光束，金色边缘 + 蓝白核心

第二行 — 命中/状态特效：
5. 小型命中：青色环形冲击波 + 火花迸发（4 帧序列横排）
6. 燃烧持续：橙色火焰图标 + 烟雾上升（4 帧循环动画）
7. 中型爆炸：橙红环形扩散 + 六边形碎片（4 帧序列）
8. 暴击爆炸：洋红色星爆 + 同心圆光环 + "CRIT!"字样（4 帧序列）

=== 设计原则 ===
- 几何感而非真实烟雾感（参考 Geometry Wars / Vampire Survivors）
- 硬边 + Bloom 光晕
- 矢量般干净的形状（环、六边形、星爆）
- 配色严格分阵营：玩家青/紫/白/金，敌方橙/红
- 每帧之间动作清晰

=== 排版 ===
- 单张 2400x1280 横版画布，纯黑背景 (#000000)
- 8 个面板，每个有清晰边框分隔
- 每个面板下方标签：
  - 中文名（白色）：基础子弹 / 散射弹 / 链电 / 轨道炮 / 命中 / 燃烧 / 爆炸 / 暴击
  - 英文小字（青色）：BASIC / SPREAD / CHAIN / RAILGUN / HIT / BURN / EXPLODE / CRIT
- 多帧动画的面板内部，帧从左到右排列

=== 严格禁止 ===
- 不要写实烟雾粒子（要几何感）
- 不要让玩家子弹和敌方子弹同色（必须冷暖对立）
- 不要 8-bit 像素风
- 不要触手 / 有机曲线
```

## §4 被动技能光环与图标

**目标**：词条卡片图标 + 玩家飞机周围的持续光环效果。

**做几次**：1 次

### §4.1 被动技能 sprite sheet

**完整 prompt（复制这一整段）**：

```
=== 视觉风格 ===
严格参照所附参考图（三职业飞机基础态）的美术风格、配色饱和度、Bloom 强度。

风格关键词：Neo-Retro Mecha、几何感、自发光 + Bloom。

=== 任务 ===
为 roguelike 词条系统设计被动技能的光环效果与图标。

=== 内容（10 个被动技能）===

光环类（挂在玩家飞机周围的持续效果，透明背景）：
1. 暴击光环：洋红色脉动星辰围绕飞机
2. 护盾光环：青色六边形护盾外壳
3. 吸血光环：暗红色血雾螺旋
4. 速度光环：白色风线延伸
5. 燃烧光环：橙色火焰粒子环

图标类（用于 UI 显示词条卡片，带六边形品质边框）：
6. 攻击力 +：青色十字剑图标
7. 射速 +：闪电符号
8. 穿透 +：箭头穿透盾牌图标
9. 子弹数 +：扇形发射图标
10. 暴击率 +：双星爆图标

=== 设计原则 ===
