## 快速开始

### 你要做什么？

| 你想生成 | 跳到章节 |

| 词条卡片图标、技能光环 | [§4 被动技能光环与图标](#4-被动技能光环与图标) |
| 大招全屏特效 | [§5 大招全屏特效](#5-大招全屏特效) |
| 敌人小怪 / 精英 / Boss | [§6 敌人三档](#6-敌人三档) |
| UI 按钮、HUD、卡片框 | [§7 UI 元素](#7-ui-元素) |
| 选角卡片背景 / 装饰素材 | [§8 选角卡片素材](#8-选角卡片素材) |

### 推荐生成顺序

1. §1 飞机本体（最先，后续武器要挂在飞机上）
2. §2 武器挂件
3. §6 敌人三档
4. §3 子弹特效
5. §4 被动光环
6. §5 大招特效
7. §7 UI 元素（最后，依赖前面的视觉风格）

### 通用工作流

```
1. 上传"美术规范参考图"作为 reference image
2. 复制对应章节的 prompt
3. 跑图，生成 4-6 张挑最满意的
4. 切图入引擎，放到 assets/arts/ 对应目录
```

### 美术规范参考图

**所有 prompt 跑图前都必须上传这张图作为 reference**：

📌 **当前规范图**：`assets/arts/characters/three_classes_concept.png`
（三职业飞机基础态对比图，2026-04-28）

理由：AI 看图比看文字理解风格更准，统一画风/配色/细节密度。

### 输出验证

跑完图后用一行命令验证 alpha：

```bash
sips -g hasAlpha 你的图.png
# yes = 真透明，可直接用
# no  = 没 alpha，需要换模型或后期抠图
```

## §1 飞机本体

**目标**：3 架飞机分别生成为独立透明 PNG，可直接放入引擎。

**做几次**：3 次（INFERNO / TEMPEST / BASTION 各一次）

### §1.1 INFERNO 烈焰

```
顶视垂直射击 roguelike 游戏的 INFERNO 烈焰战机基础态独立 sprite，
风格严格参照所附参考图。透明背景 PNG，居中构图。

机身设计：
- 紧凑子弹形机身，强烈纵向流线感
- 机首平台只装基础 2 根重炮管（升级后会动态加炮管到 8 根，要留挂载空间）
- 机身两侧"龙鼻孔"散热口（橙红发光，过载机制可视化）
- 顶部圆形过热核心（暗红脉动）
- 配色：纯黑装甲 (#1a1a1a) + 暗橙红装甲条 (#a83a1a) + 炽橙发光 (#ff6a1a)
- 双引擎橙黄尾焰
- 视觉重量中等，比喻"紧凑的火焰炮艇"

技术要求：
- 顶视 90 度俯视，机首朝上
- 1024×1024，飞机居中占 60-70%
- 缩到 64×64 剪影仍可分辨
- 自发光元素配 Bloom + rim light
- 真透明 alpha PNG，主体外像素 alpha=0
```

### §1.2 TEMPEST 风暴

```
顶视垂直射击 roguelike 游戏的 TEMPEST 风暴战机基础态独立 sprite，
风格严格参照所附参考图。透明背景 PNG，居中构图。

机身设计：
- 紫色晶体战机美学，不对称几何机身
- 中央菱形晶体核心暗淡（待激活，电荷资源可视化）
- 基础态没有浮游电球（升级后会动态出现 1-5 个，机身周围 4 方向要留挂载空间）
- 配色：深紫 (#3a0a5a) + 银色边缘 + 暗紫核心
- 紫色等离子双引擎
- 视觉重量最轻，最灵巧

技术要求：
- 顶视 90 度俯视，机首朝上
- 1024×1024，飞机居中占 60-70%
- 缩到 64×64 剪影仍可分辨
- 自发光元素配 Bloom + rim light
- 真透明 alpha PNG，主体外像素 alpha=0

不要画浮游电球（升级才有）。
```

### §1.3 BASTION 堡垒

```
顶视垂直射击 roguelike 游戏的 BASTION 堡垒战机基础态独立 sprite，
风格严格参照所附参考图。透明背景 PNG，居中构图。

机身设计：
- 厚重战舰形机身，宽大装甲板
- 中央一根大型主炮（轨道炮），Lv.1 必须是暗淡未充能状态
- 双副炮简洁无装饰
- 机身两侧留出挂载副炮塔/导弹舱的空间
- 配色：枪灰 (#3a3a4a) + 金色装甲条 (#d4a520) + 中央能量核心暗淡白
- 深红色厚重双引擎
- 视觉重量最厚重，比喻"缓慢的攻城炮舰"

技术要求：
- 顶视 90 度俯视，机首朝上
- 1024×1024，飞机居中占 60-70%
- 缩到 64×64 剪影仍可分辨
- 自发光元素配 Bloom + rim light
- 真透明 alpha PNG，主体外像素 alpha=0

不要画主炮充能发光（基础态是暗淡）。
```

## §2 武器挂件 sprite 库

**目标**：词条升级武器时动态挂载到飞机上的 sprite。

**做几次**：1 次（5 武器 × 5 等级 = 25 个 sprite 网格大图）

### §2.1 武器升级 sprite sheet

```
为顶视垂直射击 roguelike 设计武器挂件 sprite sheet，
风格严格参照所附参考图（Neo-Retro Mecha 顶视科幻机甲）。
所有武器顶视 90 度，炮口朝上。

布局：5 行 × 5 列网格，2400×2400，纯黑背景 (#000000)，每格 480×480。

【行 1】INFERNO 散射炮（橙色，配色同 INFERNO）
- Lv.1: 1 根短炮管
- Lv.2: 2 根并排
- Lv.3: 3 根并排，中间稍长
- Lv.4: 5 根扇形排列（小角度）
- Lv.5: 8 根扇形重炮管，全部炽烈发光，散热鳍片可见

【行 2】TEMPEST 浮游闪电球（紫色）
- Lv.1: 1 个紫色小球，微弱电弧
- Lv.2: 2 个球，电弧加粗
- Lv.3: 3 个球，电弧形成三角网
- Lv.4: 4 个球围绕中心，电弧密集
- Lv.5: 5 个球+中心核+全身电弧网，紫色雷霆光晕

【行 3】BASTION 充能轨道炮（金白色）
- Lv.1: 单管，暗淡白色能量核
- Lv.2: 双管，半亮能量核
- Lv.3: 双管 + 副炮塔×2
- Lv.4: 三管，金色装饰
- Lv.5: 三管 + 4 副炮塔，主炮拉出强烈白色光柱

【行 4】副武器：追踪导弹（通用解锁）
- Lv.1: 单导弹舱（侧挂式）
- Lv.2: 双导弹舱
- Lv.3: 双导弹舱 + 装填指示灯
- Lv.4: 三导弹舱
- Lv.5: 四导弹舱 + 中央指挥模块

【行 5】副武器：穿透激光（通用解锁）
- Lv.1: 小型激光发射器
- Lv.2: 中型
- Lv.3: 大型，能量聚焦点明显
- Lv.4: 双联装
- Lv.5: 三联装，前端能量聚焦炽烈

标签：
- 行左侧（青色英文+白色中文）：SPREAD CANNON / 散射炮 ｜ ORBIT ORBS / 闪电球
  ｜ RAILGUN / 轨道炮 ｜ MISSILES / 追踪导弹 ｜ LASER / 穿透激光
- 列顶部（白色）：Lv.1 / Lv.2 / Lv.3 / Lv.4 / Lv.5

约束：
- 同一武器 5 级保持中心对齐和相同挂载点位置
- 每级必须明显升级（不要变化太小）
```

### §2 输出文件

```
assets/arts/weapons/weapons_sheet.png
（后续切成单个 sprite）
```

## §3 子弹与命中特效

**目标**：玩家子弹、敌人子弹、爆炸、命中等动效。

### §3.1 子弹与特效 sprite sheet

```
为顶视垂直射击游戏生成子弹与命中特效 sprite sheet，

- 英文青色：BASIC / SPREAD / CHAIN / RAILGUN / HIT / BURN / EXPLODE / CRIT

约束：
- 玩家子弹冷色（青/紫/白/金），敌方子弹暖色（橙/红），不可混淆
- 多帧动画从左到右排列，每帧动作清晰
- 几何感而非写实烟雾

## §8 选角卡片素材

**目标**：选角面板（炉石风格）的卡片装饰素材。代码已用 Graphics 画了基础卡片框，
此章节素材用于**进一步美化**——把简陋的矩形卡片升级成有质感的"职业徽章卡"。

**优先级**：可选。代码层面已经能跑，先把游戏跑通再做这个。

**做几次**：1 次（单张大图，含所有卡片元素）

### §8.1 卡片素材 sprite sheet

```
为 roguelike 飞行射击手游设计选角卡片装饰素材，参考炉石传说卡片视觉语言：
强金属边框、深色底纹、职业色高光，但要符合本作 Neo-Retro Mecha 科幻风格
（不是奇幻 / 卡通 / 厚涂油画）。
风格参照所附参考图（三职业飞机基础态对比图）。

布局：单张 2400×1600，纯黑背景 (#000000)，分两区：

【上半区：通用卡片框 3 种状态，每个 400×600】
1. 卡片底框（无职业色）：
   - 深黑暗灰金属底（#0a0a14）
   - 切角矩形（科幻军事风，不要圆角）
   - 边缘有细密机械刻槽
   - 顶部留出"职业色横条"位置（不画颜色，留空槽）
   - 中间留出"立绘"圆形/六边形开口
   - 底部留出"中文名 + 英文 codename + tagline"文字位置
2. 同 1，但**选中态**：金属边缘多一圈青色 (#00e5ff) 发光描边
3. 同 1，但**禁用态**：整体灰度，描边失光

【下半区：3 个职业的"色横条 + 装饰"，每个 400×120】
